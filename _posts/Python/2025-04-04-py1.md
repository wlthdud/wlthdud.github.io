### 파이썬 언어의 특징  

**1. 코드의 단순성과 표준성**

파이썬의 다른 객체 지향 언어를 사용하는 코드의 구문보다 간결하고 간단하며 규칙적임  

서술문들은 줄로 끝나고, 블록 구조는 들여쓰기 형태로 나타남  
<br>
   
**2. 높은 이식성**

파이썬은 플랫폼이 독립적이면서 모든 OS 환경에 연동됨  
<br>

**3. 객체 지향 언어**

파이썬은 클래스 추상화와 함수형 추상화를 제공하는 객체 지향 언어이자 함수형 프로그래밍 언어로 코루틴(Coroutine)을 지원하는 언어  
<br>

**4. 확장 가능한 라이브러리**

파이썬은 방대한 표준 라이브러리를 제공함  

정규 표현식, 자동 문서 생성, 유닛 테스트, GUI 등 Python 패키지들을 지속적으로 업데이트하며 Python Package Index를 제공함  
<br>
<br>

### 파이썬의 단점

**1. GIL 문제**

현재 cPython의 Interpreter에서 동시에 오직 하나의 스레드만 동작하도록 허용하면서 생기는 문제  
<br>

**2. 다중 스레드(Multi-Thread)의 성능 저하**

cPython에서는 다중 스레드보다 다중 프로세스의 사용을 권장함  

다중 프로세싱 패키지를 제공해 스레드와 비슷하게 구현할 수 있도록 지원함  
<br>

**3. 모바일 플랫폼 라이브러리가 없음**  
<br>
<br>

### 파이썬으로 할 수 있는 일  

**◦ 시스템 유틸리티 제작**

파이썬은 운영체제의 시스템 명령어들을 이용할 수 있는 각종 도구를 갖추고 있어서 여러 시스템 유틸리티를 만들기 좋음  
<br>

**◦ GUI 프로그래밍**

GUI(Graphic User Interface) 프로그래밍이란 화면을 보며 마우스나 키보드로 조작할 수 있는 프로그램을 만드는 것. 파이썬은 이 GUI 프로그래밍을 만드는 것이 쉬움  
<br>

**◦ 웹프로그래밍**  
<br>

**◦ 데이터베이스 프로그래밍**

파이썬은 Sybase, Infomix, Oracle, MySQL, PostgreSQL 등의 데이터베이스에 접근할 수 있게 해주는 도구들을 제공  
<br>

**◦ 데이터 분석**

아직 데이터 분석은 R이라는 언어를 많이 사용하지만, 파이썬으로 만들어진 판다스라는 모듈을 사용하는 경우가 늘고 있음  
<br>

**◦ 사물 인터넷**

라즈베리 파이(Raspberry Pi)는 리눅스 기반의 아주 작은 컴퓨터로 파이썬은 이를 제어하는 도구로 사용  
<br>
<br>

### 파이썬으로 할 수 없는 일

**◦ 시스템과 밀접한 프로그래밍 언어**

파이썬으로 도스나 리눅스 같은 운영체제, 엄청난 횟수의 반복과 연산을 필요로 하는 프로그램 또는 데이터 압축 알고리즘 개발 프로그램 등을 만드는 것은 어려움  

대단히 빠른 속도를 요구하거나 하드웨어를 직접 건드려야 하는 프로그램에는 어울리지 않음  
<br>

**◦ 모바일 프로그래밍**  

파이썬은 구글이 가장 애용하는 언어지만, 안드로이드/아이폰 앱을 개발하는 것은 아직 어려움
